# 主题使用与配置

本章将会让你学会使用与配置 Hexo 主题，是一个比较重要的部分。

明确你搭建博客的初衷，如果只是为了好玩，请随意。如果是为了写博客，那就赶紧开始。折腾主题是使用 Hexo 最大的原罪。有时候没得选才是最好的选择。

名家之博，不假于主题花哨，而得知于内容。

—— 鲁迅 Hexo 之禅
