## 建立网站

安装好 Hexo 后即可建立你的网站。

首先建立一个文件夹。

```bash
mkdir <your_blog_name>  # 建立你的网站根目录，名字可以自己修改
```

然后往这个文件夹里安装 Hexo。

```bash
hexo init <your_blog_name>
```

这样就建立好了！

## 配置 Hexo

没错，就是在 Hexo 根目录 `_config.yml` 这个文件里配置！

### 网站

就是这部分：

```yaml
# Site
title:            
subtitle: 
description: 
keywords:
author: 
language: 
timezone: 
```
